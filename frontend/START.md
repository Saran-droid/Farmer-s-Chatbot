# Quick Start Guide

## 🚀 Running the Frontend

### Development Mode

```bash
cd frontend
npm run dev
```

The app will be available at **http://localhost:3000**

### Build for Production

```bash
npm run build
npm run preview
```

## 📋 Prerequisites

Make sure the backend is running on **http://localhost:8000**

```bash
# In the project root
cd backend
python -m uvicorn main:app --reload
```
